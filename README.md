# Bank Card Management System

## Запуск проекта

### Через Docker:
docker-compose up -d

### Локально:
mvn spring-boot:run

## Тестовые данные:
- ADMIN: admin@bank.com / admin123
- USER: user@bank.com / user123

## API Endpoints:
- Swagger: http://localhost:8080/swagger-ui.html
- API: http://localhost:8080/api
